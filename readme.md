## WELCOME TO OUR PROJECT "BACHAO"!!!

## Project Description:

Our project is a **Hero Response System** designed to manage emergency situations in a specific area. In many places, different types of problems can occur at the same time, such as crimes, fires, or accidents. The system helps manage these situations by assigning the appropriate hero to respond.

The system includes three types of heroes:
- **Medic** – responds to medical emergencies and accidents
- **Police/Hero** – responds to crimes
- **Firefighter** – responds to fire incidents

The user can control which hero is sent to which location depending on the type of emergency. The system also includes **hero limitations**, meaning only a limited number of heroes are available at a time. If multiple incidents occur simultaneously, the system helps decide which hero should handle each situation. Sending a hero to any scene makes the number of hero count decrease by 1.

Additionally, the current situations and events happening in the area are displayed on the screen so the user can monitor and manage emergencies efficiently.

## Features
- Multiple emergency types (Crime, Fire, Accident)
- Different hero units (Medic, Police, Firefighter)
- User-controlled hero assignment
- Limited hero resources
- On-screen display of ongoing situations
